# Naomi Ivie

Personal portfolio for Naomi Ivie: founder of Education for Equality and builder of voice-first AI products, developer tools, and personal systems.

Live: https://naomiivie.vercel.app

## What the site highlights

- Sabi, the private flagship: a phone-call AI tutor for Nigerian children without internet or smartphones.
- Dactyl: bidirectional ASL translation glasses.
- Sage Mail: voice-first Gmail agent.
- Go: an iOS commitment device with geofence and payment consequences.
- Kai: adaptive focus coach.
- Ed.it: desktop AI video editing agent.
- Bethel, Rings, and Adjutant as additional product systems.

## Stack

Next.js 16, TypeScript, Tailwind v4, Inter, Instrument Serif, and Vercel.

## Local development

```bash
npm install
npm run dev
```
